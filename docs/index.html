FAQ

Perguntas frequentes

Meus snippets ficam armazenados onde?

Todos os dados ficam localmente no seu browser via chrome.storage.local. Nenhuma informação é enviada para servidores externos. Suas chaves de API (GitHub, OpenAI) também ficam apenas no seu dispositivo.

Posso fazer backup dos meus snippets?

Sim! Na página de Configurações (ícone ⚙ no popup) há uma seção "Dados" com opções para exportar todos os snippets como JSON e importar de volta quando precisar.

O botão "Executar" funciona em qualquer site?

Funciona na maioria das páginas web. A extensão usa chrome.debugger para executar o código, o que bypassa restrições de CSP. Páginas do sistema como chrome:// não são acessíveis.

Preciso de conta para usar?

Não. O SnippetFocus funciona imediatamente após a instalação, sem cadastro ou login. As integrações com GitHub e OpenAI são opcionais e requerem apenas sua chave de API nas configurações.

A extensão funciona em outros browsers?

Atualmente o SnippetFocus é otimizado para Google Chrome e browsers baseados em Chromium (Edge, Brave, Opera). Suporte ao Firefox está nos planos.

Como o Modo Foco funciona?

O Modo Foco detecta automaticamente o conteúdo principal da página (elementos article, main, etc.) e exibe apenas esse conteúdo em sobreposição de tela cheia com tipografia otimizada. Pressione Alt+F para ativar ou Esc para fechar.
